# CNG Bharat Admin Panel

Separate admin website for managing fuel stations.

## Quick Start

```bash
npm install
npm run dev
```

Visit: http://localhost:3001

## Default Login
- Email: `admin@cngbharat.com`
- Password: `Admin@123`
